# Available Spam Detection Model Languages

When initializing a Spam Detection model, you can specify the language of the pre-trained model that 
you want to perform inference or training with, by passing the `language` argument. For example, if 
you want to perform inference with a Spanish pre-trained model, you should initialize the 
`spam_detection` object as follows:

```python
from artifex import Artifex

spam_detection = Artifex().spam_detection(language="spanish")
```

Artifex supports performing inference and training Spam Detection models in the following languages:

- English (`language="english"`)
- Spanish (`language="spanish"`)
- German (`language="german"`)

If no language is specified, `language="english"` will be used by default.

## When to specify the model language?

Specifying the model language is only useful when 
[performing inference](/artifex/spam-detection/inference) or [training](/artifex/spam-detection/train) with a 
pre-trained model. If you are loading a previously trained model with 
[.load()](/artifex/spam-detection/load), the `language` argument won't really matter, as the model will 
be loaded with the same language it was trained with.
