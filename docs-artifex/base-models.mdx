# Base models

All tasks that can be tackled with Artifex can be performed in two ways:

1. Using **default models** that are pre-trained and ready to use for inference, but are not 
optimized for your specific use case.
2. **Fine-tuning a default model** to create a task-specific model that is better suited for your needs;
fine-tuning **does not** require any training data, as Artifex generates synthetic data 
on-the-fly, based on your instructions.

All default models are open-source models trained by us with the Artifex library itself (meta 😁),
are small in size (around 0.1B parameters and 500Mb), can run on CPU for inference and 
fine-tuning and are hosted on our [Hugging Face page](https://huggingface.co/tanaos).

Below is a list of all the available tasks in Artifex, along with the link to the default model 
for each task.

| Task | Available Languages | Default Model |
|------|---------------------|---------------|
| Text Classification | English | No default model — must be trained |
| Guardrail | English, Spanish, German | [tanaos/tanaos-guardrail-v1](https://huggingface.co/tanaos/tanaos-guardrail-v1) (English version, see page for the other languages) |
| Intent Classification | English | [tanaos/tanaos-intent-classifier-v1](https://huggingface.co/tanaos/tanaos-intent-classifier-v1) |
| Reranker | English | [cross-encoder/mmarco-mMiniLMv2-L12-H384-v1](https://huggingface.co/cross-encoder/mmarco-mMiniLMv2-L12-H384-v1) |
| Sentiment Analysis | English | [tanaos/tanaos-sentiment-analysis-v1](https://huggingface.co/tanaos/tanaos-sentiment-analysis-v1) |
| Emotion Detection | English | [tanaos/tanaos-emotion-detection-v1](https://huggingface.co/tanaos/tanaos-emotion-detection-v1) |
| Named Entity Recognition | English | [tanaos/tanaos-NER-v1](https://huggingface.co/tanaos/tanaos-NER-v1) |
| Text Anonymization | English | [tanaos/tanaos-text-anonymizer-v1](https://huggingface.co/tanaos/tanaos-text-anonymizer-v1) |
| Spam Detection | English, Spanish, German, Italian | [tanaos/tanaos-spam-detection-v1](https://huggingface.co/tanaos/tanaos-spam-detection-v1) (English version, see page for the other languages) |
