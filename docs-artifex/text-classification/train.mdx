        # `text_classification.train()`

        Trains a Text Classification model on user-specified classes and domain. 
		If `train_dataset_path` is not provided, synthetic training data will be generated for the
		specified `domain`, `classes` and `language`. If `train_dataset_path` is provided, the model 
		will be trained on the dataset located at the specified path instead, and all arguments 
		related to synthetic data generation is ignored.

		<InfoBox type='info'>
			Unlike most other tasks that can be performed with Artifex, Text Classification **does 
			not** come with a pre-trained model that can be used out-of-the-box. This is because
			Text Classification is a general task that can be applied to a wide variety of domains
			and use-cases, each requiring different classes to be recognized. Therefore, users
			need to train their own Text Classification models using this method.
		</InfoBox>

		<APIDescriptionSection title='Arguments'>
			- **domain** <ParameterTypeBadge>str</ParameterTypeBadge> <OptionalParameterBadge />\
				A string which specifies the domain or area that the model will be specialized in.
				If `train_dataset_path` is not provided, this argument is required. If 
				`train_dataset_path` is provided, this argument is ignored.
			- **classes** <ParameterTypeBadge>dict[str, str]</ParameterTypeBadge> <OptionalParameterBadge />\
				A dictionary, where each key is the name of a class (must be maximum 20 characters with no
				spaces), and each value is the description of that class.
				If `train_dataset_path` is not provided, this argument is required. If 
				`train_dataset_path` is provided, this argument is ignored.
			- **language** <ParameterTypeBadge>str</ParameterTypeBadge> <OptionalParameterBadge /> <DefaultParameterBadge value='english' />\
				A string which specifies the language that the model will be trained to understand and perform inference in.
			- **output_path** <ParameterTypeBadge>str</ParameterTypeBadge> <OptionalParameterBadge />\
				A string which specifies the path where the output files will be generated. 
				The output files consist of:
				- The training dataset
				- The output model `safetensor` and configuration files
			- **num_samples** <ParameterTypeBadge>int</ParameterTypeBadge> <OptionalParameterBadge /> <DefaultParameterBadge value='500' />\
				An integer which specifies the number of datapoints that the synthetic training dataset should consist of, and 
				that the model will be trained on. The maximum number of datapoints you can train your model on depends on 
				whether you are on a free or paid plan. If `train_dataset_path` is provided, this 
				argument is ignored.
			- **num_epochs** <ParameterTypeBadge>str</ParameterTypeBadge> <OptionalParameterBadge /> <DefaultParameterBadge value='3' />\
				An integer which specifies the number of epochs to train the model for.
			- **device** <ParameterTypeBadge>int</ParameterTypeBadge> <OptionalParameterBadge />\
				A string which specifies the device to use. It can be one of the following:
				- `0` - Use the first GPU available
				- `1` - Use the second GPU available
				- ...
				- `-1` - Use the CPU
				
				If not specified, the method will automatically select the GPU if available, otherwise it will use 
				the CPU.
			- **disable_logging** <ParameterTypeBadge>bool</ParameterTypeBadge> <OptionalParameterBadge />\
				If set to `True`, disables inference metric logging for this call. By default, logging is enabled.
			- **train_dataset_path** <ParameterTypeBadge>str</ParameterTypeBadge> <OptionalParameterBadge />\
				A string which specifies the path to a custom training dataset to train the model on. If this argument is provided,
				synthetic data will not be generated, all arguments related to synthetic data generation (i.e. `language`, 
				`num_samples` etc.) are ignored and the model will be trained on the provided dataset instead. 
		</APIDescriptionSection>

from artifex import Artifex

text_classification = Artifex().text_classification()

text_classification.train(
    domain="chatbot conversations",
    language="english",
    classes={
        "politics": "Messages related to political topics and discussions.",
        "sports": "Messages related to sports events and activities.",
        "technology": "Messages about technology, gadgets, and software.",
        "entertainment": "Messages about movies, music, and other entertainment forms.",
        "health": "Messages related to health, wellness, and medical topics.",
    }
)
