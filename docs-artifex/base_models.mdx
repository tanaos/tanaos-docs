# Base models

All tasks that can be tackled with Artifex can be performed in two ways:

1. Using **default models** that are pre-trained and ready to use for inference, but are not 
optimized for your specific use case.
2. **Fine-tuning a default model** to create a task-specific model that is better suited for your needs;
fine-tuning **does not** require any training data, as Artifex generates synthetic data 
on-the-fly, based on your instructions.
