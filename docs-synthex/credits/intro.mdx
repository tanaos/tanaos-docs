# Credits

The `Credits` API lets you perform operations on the credits balance you have on 
the <a href={PLATFORM_BASE_URL} target='_blank' rel='noreferrer'>Tanaos platform</a>.

The credits are automatically used to pay for any paid service you use on the platform
or through the API.
