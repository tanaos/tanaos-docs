# Action Result

<div className='responsive-container'>
	<div className='text-container'>
        An `ActionResult` is an object which contains the result of an action. It is used to clarify the outcome of services 
        whose effect may be ambiguous or not immediately obvious from the request itself, such as services that generate data or 
        perform long-running tasks.

        <APIDescriptionSection title='Content'>
            - **success** <ParameterTypeBadge>bool</ParameterTypeBadge>\
            Indicates whether the action was successful or not.
            - **message** <ParameterTypeBadge>str</ParameterTypeBadge>\
            A message describing the result of the action. This can be used to provide additional information about the success or failure of the action.
        </APIDescriptionSection>
