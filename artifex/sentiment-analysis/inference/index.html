sentiment_analysis()

Perform inference with a Sentiment Analysis model, on a single input string or a list of input strings. For each input string, this will return the predicted sentiment label, which is one of the classes that the model was trained on.

If no model is loaded, the base Sentiment Analysis model tanaos/tanaos-sentiment-analysis-v1 will be used by default. For more information on the base Sentiment Analysis model, see the model's Hugging Face page.

Arguments

  • text

    str | list[str]

    A string or a list of strings to classify. The model will return a label for each input string.

  • device

    int
    optional

    A string which specifies the device to use. It can be one of the following:

    • 0 - Use the first GPU available
    • 1 - Use the second GPU available
    • ...
    • -1 - Use the CPU

    If not specified, the method will automatically select the GPU if available, otherwise it will use the CPU.

Response

A list[dict], each dict containing a "label" key with the predicted sentiment label for the corresponding input string, and a "score" key with a float value representing the model's confidence in its prediction.

  • Python
from artifex import Artifex

sentiment_analysis = Artifex().sentiment_analysis

label = sentiment_analysis("While the battery life is average, the camera quality is good.")
print(label)
  • Response
[{'label': 'neutral', 'score': 0.9254}]
